# Wazanerlingara

A project consisting of AI components,a mobile app, a client application, and a server backend.

## Structure

- [ai/](ai/) - AI-related logic and components.
- [client/](client/) - Frontend/client application.
- [mobile/](mobile/) - Mobile/mobile app with react-native ([Expo](https://expo.dev/)).
- [server/](server/) - Backend/server logic.

## Prerequisites

Before getting started, ensure you have the following installed:

- **Node.js** 18+ ([Download](https://nodejs.org/)) - includes **npm** (used for mobile development)
- **Yarn** package manager (required for frontend development):
  ```bash
  npm install -g yarn
  ```
- **NestJS CLI** (required for backend development):
  ```bash
  npm i -g @nestjs/cli
  ```

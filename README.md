# Wazanerlingara

A project consisting of AI components, a client application, and a server backend.

## Structure

- [ai/](ai/) - AI-related logic and components.
- [client/](client/) - Frontend/client application.
- [server/](server/) - Backend/server logic.

## Prerequisites

Before getting started, ensure you have the following installed:

- **Node.js** 18+ ([Download](https://nodejs.org/))
- **Yarn** package manager:
  ```bash
  npm install -g yarn
  ```
- **NestJS CLI** (required for backend development):
  ```bash
  npm i -g @nestjs/cli
  ```

## Getting Started

Refer to the README files in each subdirectory for specific instructions.

# CMS Backend - Medium-like Content Management System

NestJS backend application for a Medium-like CMS platform.

## Features

- Authentication & Authorization
- User Management
- Articles/Posts Management
- Comments System
- Tags & Categories
- Media Upload
- Followers/Following System

## Installation

```bash
yarn install
```

## Running the app

```bash
# development
yarn start:dev

# production mode
yarn start:prod
```

## Test

```bash
# unit tests
yarn test

# e2e tests
yarn test:e2e

# test coverage
yarn test:cov
```

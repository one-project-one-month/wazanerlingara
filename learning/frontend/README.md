# CMS Frontend - Medium-like Content Management System

Angular 20 frontend application for a Medium-like CMS platform.

## Features

- Authentication (Login/Register)
- Article Management (Create, Read, Update, Delete)
- User Profile Management
- Dashboard
- Responsive Design

## Installation

```bash
yarn install
```

## Development server

```bash
yarn start
```

Navigate to `http://localhost:4200/`

## Build

```bash
yarn build
```

## Running unit tests

```bash
yarn test
```
